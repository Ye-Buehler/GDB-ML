============
Installation
============

From sources
------------

The sources for GDB-ML can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Ye-Buehler/gdb_ml

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/Ye-Buehler/gdb_ml/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Ye-Buehler/gdb_ml
.. _tarball: https://github.com/Ye-Buehler/gdb_ml/tarball/master
